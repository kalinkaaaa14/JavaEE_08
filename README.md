# JavaEE_08
Потрібно додати валідацію до книжкового сервісу

1. При створені книжки потрібно валідувати
   - ISBN. як валідувати ISBN можна глянути тут https://uk.wikipedia.org/wiki/ISBN#%D0%A1%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%B0_ISBN
   - те що назва книги та автом були введені користувачем
2. При реєстрації нового користувача
  - користувач ввів валідний логін(лише латинські букви та цифри)
  - пароль користувача від 8 до 20 символів
